## Objective
Create a Python script that connects to a MySQL server, creates a database and table, and populates it with user data from a CSV file.

## Files
- `seed.py`: Script containing database setup and data insertion functions.
- `user_data.csv`: Sample dataset.
- `0-main.py`: Test script provided by ALX.

## Functions
- `connect_db()`: Connect to MySQL server.
- `create_database(connection)`: Create `ALX_prodev` database.
- `connect_to_prodev()`: Connect to the `ALX_prodev` database.
- `create_table(connection)`: Create `user_data` table.
- `insert_data(connection, data)`: Insert CSV data.
